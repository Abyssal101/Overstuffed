# Overstuffed
A Minecraft weight gain mod.
# Features
- Dynamic systems with visual bars:
  - Stuffed.
  - Weight.
- Effects based on weight:
  - Momemtum.
  - Other effects TBD.
- CPM compatibility:
  - Value layers for Stuffed and Weight bars.
# Commands
- Clear Layers
- Set Layer
- Set Maximum Weight
- Set Minimum Weight
- View layers
